How it works

From zero to multi-repo workspace in seconds.

Define your repos once, then create isolated workspaces with a single command. Every repo gets its own branch, worktree, and VS Code workspace file.

Quick Start

Up and running in under a minute.

Install, create a config, sync your repos, and start a new workspace. Both hike and hk are installed automatically.

~/projects
$ hk init
Created hike.yaml

$ hk sync
Syncing group "platform" (3 repos)...

  shared-lib: cloned
  api: cloned

$ hk platform
Creating "platform-bold-cedar"...
Project "platform-bold-cedar" created.

$ cd platform-bold-cedar
$ hk status

Configuration

One YAML file.
That's it.

Define your git org, groups, repos, and hooks. Short names expand automatically. Override anything per-repo.

Reference

Commands

Every command works with hike or hk. Commands that take a project name auto-detect it from cwd.

CommandDescription
hk [group] [name] Create a new project with worktrees for all repos in the group. First arg is matched against known groups; if no match, treated as the project name with the default group. Omit name for a random one. Flags: -c <color> sets VS Code title bar color. -c alone picks a random color. -g <group> -n <name>
hk init Create an annotated hike.yaml config file in the current directory. Prompts before overwriting an existing file.
hk sync Clone any missing repos into .hike/ and hard-reset existing ones to origin/HEAD. Run after editing config to add new repos. All clones run in parallel. Warning: performs git reset --hard on .hike/ repos. Never work in .hike/ directly.
hk pull Run git pull --ff-only on every repo in the project. Parallel across all repos.
hk push Run git push on every repo in the project. Parallel across all repos.
hk status Show branch, dirty state, and ahead/behind for each repo in the project.
hk delete Remove a project and clean up all git worktrees and branches.
hk list List all projects in the current hike root.
hk update Self-update to the latest release. No sudo needed after initial install.
