hike

A fast CLI for managing multi-repo workspaces with git worktrees.

$ curl -fsSL https://raw.githubusercontent.com/zuplo/hike/main/install.sh | sh
GitHub Releases

Instant workspaces

Create isolated workspaces across all your repos with a single command. Each gets its own branch and VS Code workspace.

Fast & parallel

Clones, syncs, pulls, pushes, and hooks all run in parallel across repos. Built in Go for speed.

Short repo names

Set your git org once and use just the repo name. Supports GitHub, GitLab, Bitbucket, and self-hosted.

Lifecycle hooks

Auto-run npm install, pnpm install, or any command after creating a project. Global, group, or repo level.

Auto-detect project

hk pull, hk push, hk status, and hk delete work from inside any project directory.

Self-updating

Checks for updates daily. Run hk update to upgrade — no sudo needed after initial install.

Quick Start

# Install
curl -fsSL https://raw.githubusercontent.com/zuplo/hike/main/install.sh | sh

# Set up a project directory
mkdir my-projects && cd my-projects
hk init                          # creates hike.yaml

# Edit hike.yaml, then clone repos
hk sync

# Create a workspace
hk platform my-feature           # -> platform-my-feature/
hk platform                      # -> platform-bold-cedar/ (random name)

# Work inside a project
cd platform-my-feature
hk pull                          # pull all repos
hk push                          # push all repos
hk status                        # git status across repos

Configuration

# hike.yaml git: org: your-org # ssh: true hooks: onCreate: pnpm install groups: platform: default: true repos: - my-app - shared-lib - repo: api branch: develop marketing: aliases: [mktg] repos: - website - cms

Commands

hk [group] [name] [-c color]   # Create a project (default command)
hk init                        # Create hike.yaml
hk sync                        # Clone & sync repos
hk pull                        # Pull all repos in project
hk push                        # Push all repos in project
hk status                      # Git status across repos
hk delete                      # Delete project & worktrees
hk list                        # List all projects
hk update                      # Self-update
